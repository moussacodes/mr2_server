## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

#satrt up docker container
sudo npm run db:dev:up

#stop docker container
sudo npm run db:dev:rm

#stop docker container, then start it, and migrate prisma
sudo npm run db:dev:restart


```
